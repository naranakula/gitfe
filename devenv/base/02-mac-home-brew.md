## mac 电脑安装软件  brew install

#### 安装命令
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
#### 官方地址
```
https://github.com/Homebrew/brew

https://docs.brew.sh/Analytics
```

## 作用
> 用来安装各种软件比如：mysql  nginx
